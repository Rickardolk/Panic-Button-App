# 🚨 Panic Button — Emergency Alert System

### Deskripsi Aplikasi

Panic Button adalah aplikasi fitur darurat yang memungkinkan seseorang untuk mengirimkan sinyal atau permintaan bantuan secara cepat dalam situasi berbahaya atau mendesak.
Aplikasi ini terintegrasi dengan perangkat IoT yang berfungsi sebagai alarm fisik ketika kondisi darurat terdeteksi atau tombol darurat diaktifkan.

---

### ⚠️ Penting

Untuk alasan keamanan dan privasi, proyek ini tidak menyertakan file konfigurasi API (seperti `google-services.json`, API key, atau kredensial lainnya) di dalam repository.
